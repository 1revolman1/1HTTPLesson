Git-tegs

Название команды Ввод в консоле Описание
Git Init $ git init Эта команда создаёт в текущем каталоге новый подкаталог с именем .git содержащий все необходимые файлы репозитория — основу Git-репозитория. На этом этапе ваш проект ещё не находится под версионным контролем.
Git Clone $ git clone [url] Используется, если вы желаете получить копию существующего репозитория Git.
Git Status $ git status Основной инструмент, используемый для определения, какие файлы в каком состоянии находятся.
Git Add $ git add Для того чтобы начать отслеживать (добавить под версионный контроль) новый файл.
Git Diff $ git diff --staged Если вы хотите посмотреть, что вы проиндексировали и что войдёт в следующий коммит.
